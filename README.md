# インストール方法
```
$ git clone git@github.com:ezawaaoi/robosys2022.git
$ cd robosys2022/
```
# plusコマンド
![test](http://github.com/ezawaaoi/robosys2022/actions/workflows/test.yml/badge.svg)
標準入力から読み込んだ数字を足します。

* 実行結果
```
$ seq 5 | ./plus
15
```
